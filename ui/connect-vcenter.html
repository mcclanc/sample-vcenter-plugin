Connect this vCenter to Tanzu Hub

In Tanzu Hub you add a vCenter under Administration > VCF Instances > Connect VCF Instance and choose vCenter (or use vCenter for the first connection). The fields below match the product form described in Connect and manage VMware Cloud Foundation (VCF) instances. This page only captures values for your runbook; it does not call Tanzu Hub.

Use the FQDN or IP that Tanzu Hub can reach (TLS on port 443). If selected, the vCenter user must have read/write permissions; otherwise read-only is enough for inventory collection. Tanzu Hub validates TLS using this certificate. Obtain it from the vCenter login page or openssl s_client -connect [vcenter-fqdn]:443 per the documentation.
