App Platform as a Service

VMware Tanzu Platform is an AI-ready, integrated private cloud Platform-as-a-Service (PaaS) designed to accelerate development, deployment, and management of secure applications.

Tanzu Platform evaluation appliance

Appliance package

Choose an .ova or .ovf from this machine. It is uploaded to the plug-in server (the host running npm start), then deployed with ovftool. Large packages can take several minutes to upload. Evaluation builds are listed on the Tanzu Platform evaluation appliances portal (download separately, then browse here).

Deploy

Enter vCenter connection and placement. The plug-in server runs VMware OVF Tool (ovftool) against vCenter using these values. Install ovftool on the server (PATH, tools/ovftool, or OVFTOOL_PATH). The OVF network label must match the package (often VM Network).
